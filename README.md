Simple Frame Assistant (SFA)

Lightweight PvP & PvE addon focused on clarity, fast reactions, and clean combat visuals.

✨ Features
🟢 Friendly Frames
Smart context-based behavior:
Solo → shows only player
Party / Dungeon → up to 5 players
Raid → supports 10 / 25+ players with auto scaling
Context-based layouts (separate positions for World, Arena, Dungeon, Raid)
Click-cast support:
Left / Right / Middle click macros
Buff & debuff tracking
Optional class-colored health bars
Option to hide Blizzard raid frames
Auto-adjusts size in large groups (BG / Raid)
🔴 Enemy Frames
Clean and minimal enemy tracking
Arena-aware (multiple enemies) vs World (target only)
Clear target highlighting
Click-cast support for CC abilities (Cyclone, Roots, etc.)
Optimized for fast readability in combat
🎯 Nameplate Enhancements
❌ Target Marker
Displays a bold red "X" above your current target’s nameplate
Designed for high visibility in PvP and chaotic fights
❗ Quest Indicator
Displays a yellow "!" above quest-related NPCs
Automatically detected via tooltip
⚔️ Simulation Mode (Advanced)
Fully simulate layouts without entering combat
Modes:
World
Arena (3v3)
Dungeon / Party
Raid 10
Raid 25
Simulates:
Roles (Healer / DPS / Tank)
Buffs & debuffs
Real layout behavior

✅ Simulation is not just preview — it directly configures real layouts
